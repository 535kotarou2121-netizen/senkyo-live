# Omni-Video Creator

このスキルは、**動画生成AI (Kling AI, Sora, Gen-3, Veo等)** のために最適化された、高品質なプロンプトを作成する専門的なAIアシスタントです。
画像生成とは異なり、**「時間経過」「物理的な動き」「カメラワーク」** に重点を置いたプロンプトを構築します。

## 主な機能

1.  **6要素フォーミュラの適用**: 動画プロンプトを論理的な6つの要素に分解し、AIの理解度を最大化します。
    *   Subject, Action, Environment, Camera, Lighting, Tech Specs
2.  **専門的カメラワークの指定**: Dolly, Truck, Orbit, Rack Focusなどの映画撮影用語を適切に使用し、映像のクオリティを上げます。
3.  **物理・時間の一貫性**: 風、重力、慣性などの物理現象を記述し、リアルな動画生成を促します。
4.  **キャラクターの一貫性保持**: 特定のキャラクターを登場させる場合、リファレンス画像との一致を強制するプロンプトを含めます。

## 使用方法

以下のコマンドでスキルを呼び出し、描きたいシーンの概要を伝えてください。

```bash
@Omni-Video Creator [シーンの説明や要望]
```

### 入力例
> "ステージ上で激しくドラムを叩く女性。カメラはダイナミックに動いて、照明は赤くて激しい点滅。"

### 出力例
1.  **Main Prompt (English)**: 専門用語を使った詳細な英語プロンプト。
2.  **Negative Prompt**: 動画特有のアーティファクト（モーフィング、静止画化など）を防ぐネガティブプロンプト。
3.  **日本語解説**: 意図した演出の解説。

## ファイル構成
- `metaprompt.md`: スキルのコアロジックとプロンプトエンジニアリングのルール。
