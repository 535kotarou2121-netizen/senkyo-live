# 高品質なAI政治MV 制作ガイド (Production Guide)

このドキュメントは、AI生成MVの品質を安定させ、「イメージ通りにいかない」問題を減らすためのノウハウ集です。

---

## 🎯 1. コンセプト設計のコツ

### 「ギャップ」を意識する
チャンネルの成功要因は**「真面目な政治家 × 意外なジャンル」**のギャップです。

| ジャンル | ギャップ効果 | 向いているシーン |
| :--- | :--- | :--- |
| ヘヴィメタル | 強い（怒り・決意） | 演説、戦う姿 |
| サイバーパンク | 中（未来感） | 政策発表、改革 |
| 80sディスコ | 強い（親しみ） | リラックスした場面 |
| 壮大なオーケストラ | 弱（王道） | 功績の振り返り |

## 🖼️ 2. 画像生成のベストプラクティス

### キャラクターの一貫性を保つ
**問題**: 同じ人物なのに、シーンごとに顔や服が変わる。

**解決策**:
1.  **リファレンス画像を固定**: 最初に1枚「基準となる画像」を生成し、それをもとにすべてのシーンを派生させる。
2.  **服装プロンプトを統一**: 全プロンプトに `wearing a dark navy suit with a red tie` のような共通の服装記述を入れる。
3.  **Seed値を活用**: 使用ツールがSeed機能に対応していれば、同じSeedを使うことで顔の一貫性を高められる。

### フォトリアルを維持する
**Prompt (English):**
```text
photorealistic, 8k resolution, shot on Canon EOS R5, cinematic color grading, natural lighting
```
**解説（日本語）**:
- `shot on Canon EOS R5`: カメラ機種を指定すると、よりリアルな質感になる傾向。
- `cinematic color grading`: 映画風のカラー調整を指示。

**Negative Prompt (English):**
```text
cartoon, anime, illustration, painting, 3d render, CGI, digital art, unreal engine
```

## 🎥 3. 動画生成のベストプラクティス

### 違和感を減らすテクニック
**問題**: AIが生成する動画で、動きが不自然（特に手や口の動き）。

**解決策**:
1.  **大きな動きを避ける**: 激しいアクションより、ゆっくりとした動き（スローモーション、ズームイン）の方がAIは得意。
2.  **顔のクローズアップを控える**: 口元のリップシンクは現状のAIでは難しいため、全身ショットやミディアムショットを多用する。
3.  **カット割りで誤魔化す**: 1カットを長くせず、3〜5秒で切り替えることで違和感を目立たせにくくする。

### リップシンク問題への対処
**現状**: AI動画のリップシンクは精度が低い。

**対処法**:
- 歌唱シーンは**口元を見せないアングル**（横顔、後ろ姿、シルエット）を使う。
- **演説シーン**は、演説している動画素材に別の音声を乗せるのではなく、静止画にズームやパンをかけた方が違和感が少ない。
- **lipsync_wav2lip** などの後処理ツールを検討する（将来の改善項目）。

## 🎵 4. 音楽生成 (Suno AI) のコツ

### 構造タグを活用する
歌詞に `[Intro]` `[Verse]` `[Chorus]` `[End]` などのタグを入れることで、曲の構成をコントロールできる。

**Prompt (English):**
```text
[Intro]
(Dramatic orchestral opening)

[Verse 1]
Standing alone against the storm...

[Chorus]
We will rise, we will fight...
```

### 繰り返しを防ぐ
Sunoは同じフレーズを繰り返すことがある。`[End]` タグを曲の最後に明示的に入れる。

## ✅ 5. 制作チェックリスト

### 制作前
- [ ] コンセプトは「ギャップ」があるか？
- [ ] リファレンス画像（基準となる1枚）は決まっているか？
- [ ] 服装や髪型のプロンプトは統一されているか？

### 制作中
- [ ] ネガティブプロンプト（cartoon, animeなど）は入っているか？
- [ ] 動画の1カットは3〜5秒以内に抑えているか？
- [ ] リップシンクが必要なシーンは口元を隠しているか？

### 制作後
- [ ] キャラクターの顔や服装は全体で統一されているか？
- [ ] 画質・色調は統一されているか？
