# 🎵 Optimize Suno Lyrics (日本語歌詞全自動最適化)

このスキルは、ユーザーから**「歌詞を作って」「Sunoで曲を作りたい」**といった依頼を受けた瞬間に発動します。
歌詞やプロンプトを提案するだけでなく、同時に**「Suno AIにそのままコピペできる最適化ファイル」**と**「人間用の歌詞カード」**を成果物として生成します。

## 発動条件 (Triggers)
**以下の文脈・依頼があった場合、ユーザーが明示的にこのスキル名を呼ばなくても、必ずこの手順を実行してください。**

1. **歌詞作成依頼**: 「歌詞を書いて」「このテーマで歌詞を作って」
2. **楽曲作成依頼**: 「Sunoで曲を作る」「AIで歌わせるプロンプトを作って」
3. **アイデア出し**: 「こんなイメージの曲にしたい（構成案作成）」

## 手順 (Procedure)

### 1. 歌詞・プロンプトの作成 (Create Config & Lyrics)
まず、ユーザーの要望に基づいて「楽曲スタイル（BPM, Genre）」と「歌詞（日本語）」を作成・提案します。

### 2. 並列ファイル生成 (Parallel File Generation)
作成した歌詞とプロンプトを元に、以下の2種類のファイルを**同時に生成**してください。

#### A: Suno AI用データ (`lyrics_optimized.txt` または `mv_storyboard.md`内)
**目的**: Suno AIが読み間違いなく、指示通りに歌うためのデータ。
- **ひらがな化**: 漢字は全て「ひらがな/カタカナ」に変換。
- **ローマ字補助**: 助詞 `(wa)`, `(e)` や読みづらい箇所にローマ字を追加。
- **構造タグ**: `[Verse]`, `[Chorus]`, `[Intro]`, `[Outro]` を完備。
- **ダイナミクス**: `[Fortissimo]`, `[Whisper]` などの指示を追加。
- **スタイル**: 先頭に `Style Prompt: Heavy Metal, BPM 160...` を記載。

#### B: 人間用歌詞カード (`lyrics.txt`)
**目的**: 人間が読むための、綺麗な歌詞カード。
- **タグ削除**: AI用の指示タグを全削除。
- **形式**: 漢字かな交じりの本来の歌詞のみ。適度な空行。
- **情報**: タイトル、作詞者（もしあれば）、スタイルプロンプト（参考用）のみ記載。

## 出力例 (Output Examples)

**ユーザー**: 「"希望の朝"というテーマでメタル曲の歌詞を作って」

**AI (あなた)**:
1. 歌詞を考案。
2. 裏で以下のファイルを作成：
   - `projects/xxx/lyrics_optimized.txt` (ひらがな・タグだらけのAI用データ)
   - `projects/xxx/lyrics.txt` (綺麗な歌詞カード)
3. ユーザーには「歌詞案を作成し、Suno用ファイルと歌詞カードも準備しました」と報告。

---
**Note:** ユーザーに「プロンプトを出して」と言われたら、チャット欄にプロンプトを表示するだけでなく、**必ずこの2つのファイル作成まで完了させてください。**
